# React TypeScript Examples

This directory contains TypeScript examples using React, demonstrating various patterns and best practices.

## Examples

### ex1-basic-component.tsx
- Functional component with TypeScript interfaces
- Optional props
- Basic JSX with conditional rendering

### ex2-class-component.tsx
- Class component with state and props typing
- Event handlers with proper typing
- State management patterns

### ex3-hooks.tsx
- useState hook with generic types
- useEffect hook with async operations
- Loading states and error handling

### ex4-context.tsx
- React Context with TypeScript
- Custom hook for context consumption
- Theme switching implementation

### ex5-custom-hooks.tsx
- Custom hook creation with TypeScript
- Generic hooks with localStorage
- Practical todo application example

### ex6-generic-components.tsx
- Generic component patterns
- Reusable list component
- Type constraints and interfaces

## Key TypeScript + React Concepts Covered

- Component prop typing with interfaces
- State typing for both functional and class components
- Event handler typing
- Generic components and hooks
- Context API with TypeScript
- Custom hooks with proper typing
- Conditional rendering with type safety
